# Reference mechanisms

Pitch Dictionary's galaxy is original project code and artwork. No CodePen code,
layout, visual composition, or asset is shipped here. These public experiments
were studied as mechanism references on 21 July 2026:

- [Starfield by Stealy](https://codepen.io/stealy/pen/vYOBgXG) — click-triggered
  velocity change and depth trails informed the idea of a brief, causal travel
  burst between terms.
- [Gravity Particles by Jay Patel](https://codepen.io/JayPatel52970/pen/MWeMxgX)
  — draggable gravity points informed direct manipulation and preserving the
  exact place where a body is grabbed.
- [Interactive Canvas Starfield by Bigsweater](https://codepen.io/bigsweater/pen/MyqZdQ)
  — pointer-responsive depth and press feedback informed the restrained parallax
  layer.
- [Traveling through Space by Chris](https://codepen.io/chrisyboy53/pen/oXZzQb)
  — projected star streaks informed the spatial cue used during word travel.

The resulting implementation uses project-authored TypeScript, deterministic
editorial relationships, DOM buttons, and a Canvas 2D renderer. It does not use
third-party runtime code from these references.
